# 📌 Project Overview

This project presents an interactive **Crime Analysis Dashboard for the United States (2025)** developed using **Excel** and **Power BI**. The dashboard helps identify major crime hotspots, compare crime patterns across regions, and provide insights into crime intensity and severity across multiple geographic levels.

The analysis focuses on incidents recorded during **2025 only** and enables users to explore crime trends through dynamic filters, maps, KPI cards, and comparative visuals.

## 🎯 Key Objectives

- Identify major crime hotspot locations using block-level and mapping data  
- Analyze crime statistics by region (North East, North West, South East, South West)  
- Evaluate incidents based on severity levels (Critical, High, Moderate, Minor)  
- Compare crime across policing and administrative boundaries  
- Support decision-making through visual summaries and drill-down analysis

# 🔄 Steps Followed

1. Cleaned raw data in Excel (removed blanks, corrected formats)  
2. Used Excel formulas and Pivot Tables for basic summaries  
3. Imported cleaned data into Power BI  
4. Created data model and DAX calculations  
5. Built interactive dashboard using charts, maps, slicers, and KPI cards  
6. Added filters and drill-down views for deeper analysis

# 🔍 Key Insights

- Theft/Other is the most frequent crime type  
- North West region records the highest crime concentration  
- Moderate severity (**8,973 cases**) is the largest category  
- Evening shift (**10,034 cases**) has the highest incidents  
- Friday is the most common day for crimes  
- Top hotspots:
  - Voting Precinct **129**
  - District **2**
  - Cluster **8**
- Non-BID areas experience more crime than business districts  
- Monthly crime levels show seasonal variation
